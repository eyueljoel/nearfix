# Product Overview

**Serviloc** is a local service marketplace platform connecting customers with service providers.

## Core Features

- **Three User Roles**: Customer, Provider, and Admin with dedicated dashboards
- **Service Request Management**: Customers create service requests; providers submit offers
- **Offer System**: Providers bid on requests; customers accept/reject offers
- **Review System**: Customers review completed services
- **Messaging System**: Real-time communication between customers and providers tied to service requests
- **Search & Discovery**: Find services by category, location, and provider

## User Workflows

**Customer Journey**:
1. Create service request with title, description, budget, and location
2. Review offers from providers
3. Accept an offer to assign a provider
4. Communicate via messaging
5. Leave a review after service completion

**Provider Journey**:
1. Browse available service requests
2. Submit offers with price and details
3. Communicate with customers
4. Complete services and receive reviews

**Admin Journey**:
1. Monitor all users, requests, offers, and reviews
2. Platform oversight and management

## Technology Context

Built with Laravel 12 and Breeze authentication, using role-based access control for multi-tenant functionality.
